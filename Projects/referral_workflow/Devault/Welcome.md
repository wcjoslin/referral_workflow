## 🛠️ How to Use This Vault

### Adding a New Feature
1. Start with [[Templates/Feature Template|Feature Template]]
2. Fill in all sections with concrete details
3. Link from [[Projects/360X Referral Workflow/Backlog & Ideas/Ideas|Ideas.md]]
4. Update [[Projects/360X Referral Workflow/Backlog & Ideas/In Progress|In Progress.md]] when work begins

### Creating a New PRD
1. Use [[Templates/PRD Template|PRD Template]]
2. Follow the structured format (Context, Goal, User Stories, AC, Technical Specs)
3. Add to the Features folder
4. Link from [[Projects/360X Referral Workflow/Features/📋 PRD Index|PRD Index]]

### Tracking Progress
- Keep [[Projects/360X Referral Workflow/Backlog & Ideas/In Progress|In Progress.md]] updated
- Mark items complete with ✅
- Move blocked items to a notes section with context

## 🔗 Vault Navigation Tips

**Obsidian Shortcuts:**
- `Ctrl+K` or `Cmd+K` — Quick file search
- `Ctrl+O` or `Cmd+O` — Open file by name
- `Ctrl+P` or `Cmd+P` — Command palette
- `Ctrl+Click` or `Cmd+Click` on a link — Opens in sidebar preview

**Linking Documents:**
- Use `[[filename]]` to create links to other notes
- Use `[[filename#section|custom text]]` for links to specific sections with custom labels
- Hover over links to preview content
