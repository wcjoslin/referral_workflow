# ICD-10 Documentation Requirements

## Policy

All specialist referrals must include at least one ICD-10 diagnosis code that clinically supports the reason for referral. This is required for:

- Insurance pre-authorization
- Accurate billing and claims processing
- Clinical decision-making by the receiving specialist

## Minimum Requirements

1. **Primary diagnosis code**: At least one ICD-10-CM code must be present in the problems/diagnoses section.
2. **Clinical relevance**: The diagnosis code(s) must be clinically related to the reason for referral.
3. **Specificity**: Codes should be as specific as possible (use 4th, 5th, 6th, and 7th characters where applicable).

## Common Referral Categories and Expected Codes

| Referral Type | Expected ICD-10 Range |
|---|---|
| Chronic Pain | M54.x, G89.x, M79.x |
| Diabetes Management | E10.x, E11.x, E13.x |
| Cardiology | I10-I15, I20-I25, I47-I49 |
| Orthopedic | M00-M99 |
| Mental Health | F01-F99 |
| Pulmonary | J00-J99 |

## Exceptions

- Emergency or urgent referrals may proceed without ICD codes, with documentation to follow within 48 hours.
- Referrals for screening/preventive services may use Z-codes (Z00-Z99).
